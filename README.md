# Site-Profissional-Joao-Gabriel-Moimaz-de-Souza
--Descrição do Projeto--

Este é o repositório do meu Site profissional, desenvolvido como um projeto de apresentação profissional. O objetivo principal é exibir minhas informações de contato, formação, experiências profissionais, e demonstrar minhas habilidades em desenvolvimento web front-end.

--Tecnologias Principais--

Front-end: HTML5, CSS3, JavaScript.

Estilização e layout: Bootstrap v4.5.2 (para Navbar e Carrossel) e CSS Media Queries (para responsividade).

Design: Tema Escuro (#070F26), Fonte Inter, Ícones Font Awesome.


--Funcionalidades Implementadas--

Página inicial:
Exibe uma introdução sobre mim e a identidade visual do site (logo, foto).

Projetos:
Exibe um carrossel ou galeria para destacar meus projetos(no caso um só), com link para o repositório no GitHub para acesso ao código-fonte.

Sobre Mim (Perfil Profissional):
Detalhe de Carreira: Apresenta a minha formação acadêmica e  experiência profissional.
Documentação Técnica: Destaca a minha capacidade de elaborar documentos técnicos (análises de risco, especificações, protocolos de qualificação).
Suporte e Testes: Fala sobre a minha experiência em condução de testes de validação de sistemas e suporte administrativo em sistemas ERP (SAP).

Contato:
Oferece um formulário de contato funcional (embora a lógica de envio não esteja visível, assume-se a intenção de uso).
Fornece acesso rápido a perfis profissionais e sociais (LinkedIn, GitHub, Instagram) através de ícones no footer.

Responsividade:
O design é totalmente responsivo, adaptando-se perfeitamente a dispositivos móveis, tablets e desktops.
